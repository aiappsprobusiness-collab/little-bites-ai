# Чеклист для диагностики проблемы "Сайт не открывается"

## Шаг 1: Проверьте, что сервер запущен

В терминале должно быть:
```
VITE v7.3.1  ready in xxx ms
➜  Local:   http://localhost:8080/
```

Если этого нет - сервер не запущен. Запустите `npm run dev`.

## Шаг 2: Откройте браузер

Перейдите на: **http://localhost:8080/**

## Шаг 3: Откройте DevTools (F12)

Нажмите F12 в браузере и проверьте:

### Вкладка Console
- Есть ли красные ошибки?
- Скопируйте все ошибки

### Вкладка Network
- Загружаются ли файлы?
- Есть ли файлы с ошибками (красные)?

## Шаг 4: Что должно произойти

1. **Если вы НЕ авторизованы:**
   - Должна открыться страница `/auth` (форма входа/регистрации)
   - Это нормально!

2. **Если вы авторизованы:**
   - Должна открыться главная страница

## Шаг 5: Возможные проблемы

### Белый экран
- Откройте Console (F12)
- Проверьте ошибки JavaScript
- Сообщите об ошибках

### Бесконечная загрузка
- Проверьте подключение к Supabase
- Проверьте Console на ошибки сети

### Ошибка "Cannot find module"
- Выполните `npm install`
- Перезапустите сервер

### Ошибка GigaChat
- Это не должно блокировать приложение
- Приложение должно работать без GigaChat

## Шаг 6: Быстрая проверка

1. Откройте http://localhost:8080/auth напрямую
2. Должна открыться страница входа
3. Если открывается - приложение работает!
