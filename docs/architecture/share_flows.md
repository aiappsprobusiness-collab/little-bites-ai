# Шаринг плана (текст и ссылки)

Коротко: что уходит в мессенджеры при «Поделиться» планом дня и недели.

## Общее

- Тексты собираются в **`src/utils/shareMenuText.ts`** (день — `buildDayMenuShareBody`, неделя — `buildWeekMenuShareBody`).
- **Рецепт (чат и карточка):** `buildRecipeShareTextForShare` в `src/utils/shareRecipeText.ts` — полная карточка (ингредиенты, шаги, совет) + короткая ссылка `/r/:shareRef` в футере; при длине > 3500 символов — fallback на `buildRecipeShareTextShort`.
- Вступление дня — **`getShareIntroText`** в `src/utils/shareDayMenuText.ts` (без бренда в каждой строке; бренд/ссылка — в CTA и футере).
- Опционально после intro — **`buildAllergyShareLine`** (аллергии профиля плана, без имени ребёнка).
- **Копирование:** `copyShareTextToClipboard` в `src/utils/shareActions.ts` — кнопка «Скопировать текст» в превью плана, на `RecipePage` и в `ChatMessage` (`share_click` с `channel=copy_link`).
- Превью плана (`MealPlanPage`): подсказка **`SHARE_MENU_PREVIEW_HINT`** (только UI, не уходит в сообщение).
- Лендинг плана (`SharedPlanPage`): social proof — «Соберите похожее меню для своей семьи за ~30 секунд».

## Неделя: формат текста

1. **Заголовок** — один из трёх вариантов (случайный при открытии превью; индекс фиксируется в state, чтобы превью совпало с отправкой):
   - «Посмотри, какое меню получилось на неделю 👇»
   - «Собрали меню на неделю — удобно и без заморочек 👇»
   - «Вот наш план питания на неделю 👇»
2. Пустая строка после заголовка.
3. **По дням** (только дни, где есть что показать):
   - строка с коротким именем дня (`Вс`, `Пн`, …);
   - далее строки приёмов пищи в порядке: завтрак → обед → ужин → перекус;
   - между днями — **пустая строка**;
   - блюда **не** склеиваются через запятую в одной строке.
4. **Строка приёма пищи:** `[эмодзи] [тип]: [название]`.
5. **Эмодзи для недельного шаринга:**
   - завтрак — 🍚
   - обед — 🥣
   - ужин — 🍲
   - перекус — 🍎
6. Если слота нет в данных — строка для этого типа не выводится.
7. **CTA перед ссылкой** — один из двух вариантов (тоже фиксируется индексом при открытии превью):
   - «🛒 Список продуктов соберётся автоматически:»
   - «👉 Открыть меню и собрать список продуктов:»
8. URL идёт **следующей строкой** после CTA (`appendShareLinkOnce`).

Источник слотов и названий — `meals` в payload недели (`slot` + `title`), см. `SharedPlanPayloadWeek` в `src/services/sharedPlan.ts`.

## Обратная совместимость

`buildWeekMenuShareBody` по-прежнему принимает строки с полем `brief` (старый формат «День — краткое описание»), если не передан массив `meals`.

## Связанные файлы

- UI: `MealPlanPage` (превью «Ваше меню», вызов share / clipboard; один раз после успешного «Собрать день» — toast с CTA «Поделиться»).
- Лендинг: `SharedPlanPage`, таблица `shared_plans`. CTA «Собрать свой план» → `/auth?mode=signup&entry_point=shared_*_plan&share_ref=...` (без промежуточного `/welcome`).
