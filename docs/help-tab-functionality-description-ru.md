# Описание функционала вкладки «Помощник рядом» (Помощь маме)

Полное описание экрана «Помощник рядом»: структура, вкладки-фильтры, карточки ситуаций, поведение кнопок и ограничения для бесплатного пользователя (Free). Документ предназначен для использования с ChatGPT или другими инструментами.

**Реализация UX и лимитов (quick chips, «Сегодня спрашивают», paywall, счётчик):** [help-tab-ux-implementation.md](architecture/help-tab-ux-implementation.md).

---

## 1. Общая структура экрана

Экран доступен по маршруту **/sos** (вкладка «Помощник» / лампочка в нижней навигации).

- **Hero-блок (шапка)**  
  - Заголовок: **«Помощник рядом»**. Справа от заголовка — **чип выбора профиля** (как в других вкладках): «Семья» или имя ребёнка. По тапу открывается выбор «Семья» / список детей.  
  - Подзаголовок: **«Что происходит с ребёнком?»**.  
  - Поле ввода «Напишите ситуацию...» и кнопка «Спросить»; под ним — текст лимита для Free («Сегодня осталось: X вопроса» или «Лимит на сегодня исчерпан»); горизонтальные чипсы быстрых ситуаций (Не хочет есть, Срыгивания, Аллергия и т.д.).  
  - Визуально: карточка с иконкой (спасательный круг), скруглённые углы, нейтральный фон.

- **Блок «Темы»**  
  - Заголовок: **«Темы»**.  
  - Подзаголовок: **«Выберите ситуацию»**.  
  - Ниже — горизонтальная полоса **кнопок-фильтров (табов)** по категориям.

- **Список карточек**  
  - Прокручиваемый список карточек. Каждая карточка — одна **ситуация/тема** (например, «Ребёнок не хочет есть — что делать?», «Аллергия или реакция — что делать?»).  
  - У карточки: иконка слева, заголовок, короткий подзаголовок, стрелка справа.  
  - Тап по карточке открывает либо **консультацию по теме** (чат с AI), либо **экран оплаты (Paywall)**, если тема только для платных пользователей.

## 2. Как работают вкладки (кнопки-фильтры)

Под текстом «Выберите ситуацию» расположены **4 кнопки**:

| Кнопка   | Код категории | Назначение |
|----------|----------------|------------|
| **Все**  | `all`          | Показать все темы (и «Питание», и «Режим», и «Аллергии»). |
| **Питание** | `feeding`   | Только темы про питание, прикорм, стул, срыгивания, отказ от еды, срочную помощь. |
| **Режим**   | `routine`   | Только темы про режим кормления и дневник питания. |
| **Аллергии** | `allergy`  | Только темы про аллергии и реакции. |

- Выбранная кнопка визуально выделена (фон primary/10, рамка primary).  
- При переключении кнопки **список карточек под ней фильтруется**: отображаются только карточки из выбранной категории.  
- Это именно **фильтр отображения**, а не отдельные экраны: один и тот же список тем, показываются только подходящие по категории.

**Примеры соответствия тем и категорий:**

- **Питание (feeding):** «Как безопасно ввести новый продукт?», «Стул малыша: норма или повод волноваться?», «Срыгивания: норма или проблема?», «Ребёнок не хочет есть — что делать?», «Когда срочно обращаться к врачу?», «Аллергия или реакция — что делать?» (аллергия дублируется и в категории «Аллергии»).  
- **Режим (routine):** «График кормления: подходит ли возрасту?», тема про дневник питания.  
- **Аллергии (allergy):** «Аллергия или реакция — что делать?».

## 3. Как работают карточки (кнопки ситуаций)

- Каждая **карточка** — это одна тема консультации (например: «Ребёнок не хочет есть — что делать?», «Аллергия или реакция — что делать?», «Стул малыша: норма или повод волноваться?», «Срыгивания: норма или проблема?», «График кормления: подходит ли возрасту?» и др.).  
- **Тап по карточке:**  
  - Если тема **доступна пользователю** (см. раздел про Free) — открывается **нижняя панель (Sheet)** с консультацией по этой теме: заголовок темы, подсказки-чипсы (короткие формулировки вопросов), поле ввода и чат с AI. Пользователь может задавать вопросы в рамках этой темы и получать ответы.  
  - Если тема **только для платных** (Premium/Trial) — открывается **Paywall** (экран предложения подписки), консультация не открывается.

На карточках тем, доступных только по подписке, отображается бейдж **«Premium»** (справа).

## 4. Функция кнопок для бесплатного пользователя (Free)

**Доступно бесплатно:**

- Просмотр Hero-блока и текста «Помощник рядом» / «Выберите ситуацию и получите рекомендации».  
- Использование **вкладок-фильтров** («Все», «Питание», «Режим», «Аллергии») — фильтрация списка карточек.  
- Просмотр **всех карточек** в списке (в том числе с бейджем «Premium»).  
- Открытие консультации (Sheet) по темам с **доступом для Free** (см. ниже).  
- В открытой консультации: просмотр чипсов-подсказок, ввод своего вопроса, отправка сообщения AI и получение ответа — **в пределах дневного лимита** (см. ограничения).

**Ограничения для Free:**

1. **Доступ к темам**  
   Часть тем помечена как **только для платных** (`requiredTier: "paid"`). При тапе по такой карточке у Free открывается **Paywall**, а не консультация.  
   Примеры тем только для Premium/Trial: «Ребёнок не хочет есть — что делать?», «Срыгивания: норма или проблема?», «График кормления: подходит ли возрасту?», тема про дневник питания.  
   Темы с `requiredTier: "free"` доступны всем: например «Как безопасно ввести новый продукт?», «Аллергия или реакция — что делать?», «Стул малыша: норма или повод волноваться?», «Когда срочно обращаться к врачу?» и др.

2. **Дневной лимит вопросов к Помощнику (help)**  
   - У Free: **2 запроса к AI в день** в рамках раздела «Помощник рядом» (по темам).  
   - Считаются только **успешные** отправки сообщений в консультации (тип запроса `sos_consultant`); на бэкенде пишется событие в `usage_events` с `feature: "help"`.  
   - Когда лимит исчерпан, при следующей попытке отправить сообщение бэкенд возвращает ошибку **LIMIT_REACHED**; в приложении показывается Paywall с текстом:  
     - Заголовок: **«Лимит на сегодня исчерпан»**.  
     - Сообщение: **«В Free доступно 2 вопроса в день. В Trial/Premium — без ограничений.»**  
   - Лимит обновляется раз в сутки (по календарному дню).  
   - Trial и Premium: **без лимита** по числу вопросов в Помощнике.

3. **Отдельно от лимита чата «Рецепты»**  
   Лимит «Помощника» (2 вопроса в день) **не связан** с лимитом генерации рецептов в чате (тоже 2 в день для Free). То есть у Free отдельно 2 запроса в день на рецепты и отдельно 2 запроса в день на консультации по темам Помощника.

Итого для Free: можно пользоваться вкладками и открывать только те темы, у которых нет бейджа «Premium»; в этих темах можно задавать вопросы AI с лимитом **2 вопроса в день**; при исчерпании лимита показывается Paywall с сообщением о лимите.

## 5. Выбор профиля (правила для ответов)

- В hero-блоке рядом с «Помощник рядом» отображается **чип выбора профиля** (тот же компонент, что на Плане, в Чате и т.д.): «Семья» или имя выбранного ребёнка.
- **Выбран конкретный профиль (ребёнок):** консультация учитывает этого ребёнка (возраст, аллергии и т.д.), как в других вкладках. В API передаётся `memberId` = id ребёнка.
- **Выбрана «Семья»** (или `selectedMemberId` = `null` / `"family"`): ответ даётся **в общем** по запросу, либо ассистент может **уточнить, о ком речь**, если это важно для рекомендации. В API передаётся `memberId: "family"`, бэкенд строит контекст для семьи/общий ответ.
- Сессия чата по теме хранится по ключу `(memberId, topicKey)`, т.е. отдельно для «Семья» и для каждого ребёнка.

## 6. Краткая сводка для промпта ChatGPT

- **Экран:** «Помощник рядом» (Hero: заголовок + «Выберите ситуацию и получите рекомендации»).  
- **Темы:** блок «Темы» + подзаголовок «Выберите ситуацию» + 4 кнопки: **Все**, **Питание**, **Режим**, **Аллергии**. Это фильтры списка карточек по категориям.  
- **Карточки:** каждая карточка — ситуация (например «Ребёнок не хочет есть — что делать?», «Аллергия или реакция — что делать?»). Тап открывает консультацию (чат по теме) или Paywall, если тема только для платных.  
- **Free:** часть тем только по подписке (тап → Paywall); в доступных темах — 2 вопроса к AI в день, при исчерпании — Paywall с текстом «Лимит на сегодня исчерпан» и «В Free доступно 2 вопроса в день. В Trial/Premium — без ограничений.»  
- **Trial/Premium:** все темы открыты, лимита по количеству вопросов в Помощнике нет.

- **Профиль:** в hero есть чип выбора профиля; выбранный ребёнок учитывается в ответах; при «Семья» — общий ответ или уточнение, о ком речь.
