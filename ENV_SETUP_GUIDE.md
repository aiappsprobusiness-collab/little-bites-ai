# Настройка .env файла для GigaChat

## Шаг 1: Найдите файл .env

Файл `.env` должен находиться в **корне проекта** (рядом с `package.json`).

Путь: `c:\Users\alesa\OneDrive\Рабочий стол\Repositories\little-bites-ai\.env`

## Шаг 2: Откройте файл .env

Откройте файл `.env` в любом текстовом редакторе (Блокнот, VS Code, Cursor и т.д.)

## Шаг 3: Добавьте настройки GigaChat

Скопируйте и вставьте следующее содержимое в файл `.env`:

```env

```

## Шаг 4: Сохраните файл

Сохраните файл (Ctrl+S)

## Шаг 5: Перезапустите dev сервер

**ВАЖНО:** После изменения `.env` файла **обязательно перезапустите** dev сервер!

1. Остановите текущий сервер: нажмите `Ctrl+C` в терминале
2. Запустите снова:
   ```bash
   npm run dev
   ```

3. На странице "Сканировать" должна работать функция анализа изображений

## Если у вас другой ключ

Если у вас есть другой GigaChat ключ (не Base64), конвертируйте его:

### В PowerShell:
```powershell
[Convert]::ToBase64String([Text.Encoding]::UTF8.GetBytes('ваш_ключ_здесь'))
```

### В браузере (консоль):
```javascript
btoa('ваш_ключ_здесь')
```

Затем замените значение `VITE_GIGACHAT_CLIENT_SECRET_KEY` в `.env` файле.

## Формат файла .env

Файл должен выглядеть так:
```
VITE_GIGACHAT_CLIENT_SECRET_KEY=ваш_base64_ключ
VITE_GIGACHAT_IS_PERSONAL=true
```

**Важно:**
- Не ставьте пробелы вокруг знака `=`
- Не используйте кавычки вокруг значений
- Каждая переменная на новой строке
- Комментарии начинаются с `#`

## Получение GigaChat ключа

Если у вас нет ключа:

1. Перейдите на https://developers.sber.ru/gigachat
2. Зарегистрируйтесь или войдите
3. Создайте новое приложение
4. Получите Client Secret Key
5. Конвертируйте в Base64 (см. выше)

## Решение проблем

### Приложение не видит переменные окружения

- Убедитесь, что файл называется именно `.env` (с точкой в начале)
- Убедитесь, что файл в корне проекта
- Перезапустите dev сервер после изменения

### Ошибка "GigaChat не настроен"

- Проверьте, что `VITE_GIGACHAT_CLIENT_SECRET_KEY` установлен
- Проверьте, что нет лишних пробелов
- Перезапустите сервер
