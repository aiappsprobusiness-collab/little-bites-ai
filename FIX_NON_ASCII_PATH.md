# Исправление ошибки с не-ASCII символами в пути

## Проблема

Ваш проект находится в папке с кириллицей:
```
C:\Users\alesa\OneDrive\Рабочий стол\Repositories\little-bites-ai
```

Android Gradle Plugin 8+ не поддерживает пути с не-ASCII символами (кириллица, китайские иероглифы и т.д.).

## Решение 1: Временное (уже применено)

Я добавил в `android/gradle.properties`:
```properties
android.overridePathCheck=true
```

Это позволит собрать проект, но **не рекомендуется** для продакшена.

## Решение 2: Переместить проект (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Создайте новую папку без кириллицы

Например:
```
C:\Projects\little-bites-ai
```

Или:
```
C:\Dev\little-bites-ai
```

### Шаг 2: Переместите проект

1. Закройте Android Studio
2. Скопируйте всю папку проекта в новое место
3. Удалите старую папку (после проверки, что все работает)

### Шаг 3: Обновите пути

1. Откройте Android Studio
2. **File** → **Open** → выберите новую папку проекта
3. Дождитесь синхронизации Gradle

### Шаг 4: Удалите временное решение

После перемещения удалите из `android/gradle.properties`:
```properties
android.overridePathCheck=true
```

## Почему лучше переместить?

- ✅ Избежите проблем в будущем
- ✅ Более стабильная работа Gradle
- ✅ Совместимость со всеми инструментами
- ✅ Меньше проблем при работе в команде

## Текущее решение

Я уже добавил `android.overridePathCheck=true` в `gradle.properties`, так что сейчас проект должен собираться.

**Попробуйте снова:**
1. В Android Studio: **File** → **Sync Project with Gradle Files**
2. **Build** → **Build APK(s)**

Но в будущем лучше переместить проект в папку без кириллицы.
