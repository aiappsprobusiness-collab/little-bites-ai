## Почему приложение может не открываться

### 1. Dev сервер не запущен

**Проверьте:**
- Запущен ли `npm run dev`?
- Видите ли вы в терминале сообщение `Local: http://localhost:8080/`?

**Решение:**
```bash
npm run dev
```

### 2. Порт 8080 занят

**Проверьте:**
- Не запущено ли другое приложение на порту 8080?

**Решение:**
- Остановите другие приложения на порту 8080
- Или измените порт в `vite.config.ts`:
```typescript
server: {
  port: 3000, // или другой свободный порт
}
```

### 3. Ошибка в браузере

**Проверьте:**
1. Откройте DevTools (F12)
2. Перейдите на вкладку Console
3. Посмотрите, есть ли красные ошибки

**Частые ошибки:**
- `Failed to fetch` - проблема с сетью или сервером
- `Cannot find module` - проблема с импортами
- `Unexpected token` - синтаксическая ошибка

### 4. Кеш браузера

**Решение:**
- Нажмите `Ctrl + Shift + R` (жесткая перезагрузка)
- Или очистите кеш браузера

### 5. Проблема с зависимостями

**Решение:**
```bash
# Удалите node_modules и переустановите
rm -rf node_modules
npm install

# Или в Windows PowerShell:
Remove-Item -Recurse -Force node_modules
npm install
```

## Пошаговая диагностика

### Шаг 1: Проверьте, запущен ли сервер

В терминале должно быть:
```
VITE v7.x.x  ready in xxx ms

➜  Local:   http://localhost:8080/
```

Если нет - запустите:
```bash
npm run dev
```

### Шаг 2: Откройте в правильном браузере

**НЕ используйте встроенный браузер Cursor!**

Откройте в обычном браузере:
- Chrome
- Firefox
- Edge

Перейдите на: `http://localhost:8080`

### Шаг 3: Проверьте консоль браузера

1. Нажмите F12
2. Откройте вкладку Console
3. Скопируйте все красные ошибки
4. Сообщите о них

### Шаг 4: Проверьте Network

1. В DevTools откройте вкладку Network
2. Обновите страницу (F5)
3. Проверьте, загружаются ли файлы (должны быть зеленые статусы)

## Если ничего не помогает

1. **Остановите dev сервер** (Ctrl+C)
2. **Очистите кеш Vite:**
   ```bash
   # Удалите папку .vite (если есть)
   rm -rf node_modules/.vite
   ```
3. **Перезапустите:**
   ```bash
   npm run dev
   ```

## Проверка работоспособности

После запуска `npm run dev` вы должны увидеть:
```
VITE v7.x.x  ready in xxx ms

➜  Local:   http://localhost:8080/
➜  Network: use --host to expose
```

Если видите это - сервер работает! Откройте `http://localhost:8080` в браузере.
